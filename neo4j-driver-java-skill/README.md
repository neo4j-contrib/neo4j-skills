# neo4j-driver-java-skill

Guides agents through connecting to Neo4j with the raw Java driver: Maven/Gradle setup, executableQuery(), session/transaction management, and error handling.

**Install:**
```bash
npx skills add https://github.com/neo4j-contrib/neo4j-skills --skill neo4j-driver-java-skill
```

Or paste this link into your coding assistant:
https://github.com/neo4j-contrib/neo4j-skills/tree/main/neo4j-driver-java-skill
