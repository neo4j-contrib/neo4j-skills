# neo4j-driver-go-skill

Guides agents through connecting to Neo4j with the Go driver: setup, ExecuteQuery(), context propagation, session/transaction management, and result serialization with AsMap().

**Install:**
```bash
npx skills add https://github.com/neo4j-contrib/neo4j-skills --skill neo4j-driver-go-skill
```

Or paste this link into your coding assistant:
https://github.com/neo4j-contrib/neo4j-skills/tree/main/neo4j-driver-go-skill
